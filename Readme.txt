node v18.17.0 (npm v9.6.7)

Backend - Node 
Frontend - React
Database - MongoDB

install package
 - npm install (do it in both Backend and Frontend Folder. it will install all dependencies which required to run project.)

Steps to start Backend
 - Go to Backend Folder
    - node index.js
 - After Start the node project First step to fetch data from third party use this API in browser it is GET -
    - http://localhost:5000/api/products/fetch-and-store

Steps to start Frontend
 - Go to Frontend Folder
    - npm start
